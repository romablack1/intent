## Описание проекта
Данное Android-приложение разработано в рамках итогового практического задания по изучению компонента `Intent`. Проект представляет собой одноэкранное приложение "Контактная книга", написанное на Kotlin с использованием современного UI-фреймворка **Jetpack Compose**. 

Главная цель приложения — демонстрация работы с неявными намерениями (Implicit Intents) для перенаправления задач пользователя в другие системные приложения.

## Реализованный функционал
На главном экране расположены четыре кнопки, каждая из которых обрабатывает отдельный неявный Intent:

1. **«Позвонить»**: Открывает стандартное приложение набора номера (DIAL) с предустановленным номером `+7 (495) 123-45-67`.
2. **«Написать email»**: Открывает почтовый клиент (SENDTO) с подставленным адресом `contact@example.com` и темой письма `Обращение`.
3. **«Показать офис на карте»**: Открывает картографическое приложение (VIEW, схема `geo:`) на координатах `60.0237, 30.2289` с установленной меткой `Наш офис`.
4. **«Поделиться контактом»**: Вызывает системное окно выбора приложения (`Intent.createChooser()`), позволяя поделиться строкой `Контакт: +7 (495) 123-45-67, contact@example.com` в мессенджерах или соцсетях.
